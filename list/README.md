# Python Program List Practice and Exercise on List

## Reverse List

```
Input:
Enter Total Size : 5

Enter 1 no: 10
Enter 2 no: 20
Enter 3 no: 30
Enter 4 no: 11
Enter 5 no: 22

Output:
Original List: 10 20 30 11 22
Reverse List : 22 11 30 20 10
```
