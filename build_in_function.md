## ord()

Return character from number(Unicode code).

```
number = ord("a")
print(number)

Output:
97
```

## chr()

Return number(Unicode code). from character

```
char = chr(97)
print(char)

Output:
a
```

## len()

Return length of object

```
name = "sudo"
nameLength = len(name)
print(nameLength)

4
```
